# Bubble Sort (The Odin Project)

## 📘 Project Overview

This project is part of **The Odin Project Ruby course**. The goal is to build a method that sorts an array using the **bubble sort algorithm**.

Bubble sort is a simple sorting method that repeatedly steps through a list, compares adjacent elements, and swaps them if they are in the wrong order.

---

## 🧠 How Bubble Sort Works

1. Go through the array from beginning to end
2. Compare each pair of adjacent elements
3. Swap them if the left element is greater than the right
4. Continue this process for the entire array (this is one "pass")
5. Repeat passes until no swaps are needed

Once a full pass is completed without any swaps, the array is sorted.

---

## 🛠️ What I Built

* A method called `bubble_sort`
* It takes an array as input
* It returns the array sorted in ascending order
